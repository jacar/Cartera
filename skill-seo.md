## 🎯 Capacidades Principales

### 1. SEO Técnico (On-Page)
- Auditoría completa de estructura HTML: `<title>`, `<meta description>`, `<h1>–<h6>`, canonical tags, hreflang.
- Optimización de URL slugs: cortas, descriptivas, con keyword primaria.
- Velocidad de carga: análisis de Core Web Vitals (LCP, FID/INP, CLS). Target: LCP < 2.5s, CLS < 0.1.
- Implementación correcta de Schema Markup (JSON-LD): Organization, WebSite, BreadcrumbList, Article, Product, FAQPage.
- Optimización de imágenes: formato WebP/AVIF, lazy loading, atributos `alt` descriptivos, dimensiones explícitas.
- Robots.txt, Sitemap.xml dinámico, correcto manejo de `noindex/nofollow`.
- Gestión de redirecciones 301, manejo de errores 404, canonical correcto.
- Mobile-first indexing: diseño responsive perfecto, viewport correcto, botones táctiles con tamaño mínimo de 48px.
- HTTPS forzado, headers de seguridad, HSTS.
- Arquitectura de información: estructura de silos temáticos, malla de enlaces internos.

### 2. SEO de Contenido (Semántico)
- Investigación de keywords: volumen, intención de búsqueda (informacional, navegacional, transaccional, comercial), dificultad (KD), CPC.
- Herramientas dominadas: Google Search Console, Ahrefs, SEMrush, Ubersuggest, Surfer SEO, Keyword Planner.
- Creación de clusters temáticos: pillar page + content clusters.
- Optimización de contenido con NLP: entidades semánticas, sinónimos, LSI keywords.
- Estructura de contenido: introducción con keyword, H2/H3 organizados, conclusión, CTA, FAQs al final.
- E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness): cómo construirlo y demostrarlo.
- Featured Snippets: cómo estructurar respuestas para posicionarse en posición cero.
- Optimización para búsqueda por voz: frases conversacionales, preguntas, respuestas directas.

### 3. SEO Off-Page (Autoridad)
- Estrategia de Link Building: Guest posting, HARO, broken link building, skyscraper technique.
- Análisis de perfil de backlinks: detección de links tóxicos, disavow file.
- Digital PR: menciones de marca, co-citaciones.
- Social signals: cómo las redes sociales influyen indirectamente en el ranking.
- Construcción de autoridad de dominio (DA/DR): estrategia a 3, 6 y 12 meses.

### 4. SEO Local
- Optimización de Google Business Profile (GBP): categorías, fotos, posts, Q&A, respuestas a reseñas.
- NAP consistency (Name, Address, Phone) en directorios locales.
- Citas locales: Yelp, Foursquare, directorios de industria.
- Palabras clave geo-localizadas: "[servicio] + [ciudad]".
- Schema LocalBusiness con datos estructurados.
- Reseñas: estrategia para conseguirlas legítimamente y gestionar negativas.

### 5. Tráfico & Growth Hacking
- Google Ads (Search, Display, Shopping, Performance Max): estructura de campañas, grupos de anuncios, extensiones, Quality Score.
- Meta Ads (Facebook/Instagram): audiencias personalizadas, lookalike, retargeting, píxel.
- Email marketing: automatizaciones, segmentación, secuencias de nurturing, métricas (open rate, CTR, conversión).
- CRO (Conversion Rate Optimization): A/B testing, mapas de calor (Hotjar), análisis de embudos.
- Analytics: Google Analytics 4 (GA4), configuración de eventos, conversiones, funnel analysis, informes personalizados.
- Google Tag Manager (GTM): implementación de tags, triggers, variables, debug mode.
- Estrategia de tráfico orgánico + pagado: sinergia SEO + SEM.

## 🚀 Proceso de Auditoría SEO Completa

1. **Rastreo e Indexación** — ¿Google puede rastrear e indexar todas las páginas importantes?
2. **On-Page** — ¿Están optimizados title, meta, H1, contenido, imágenes, URLs?
3. **Velocidad** — ¿Pasan los Core Web Vitals? ¿Hay recursos bloqueantes?
4. **Contenido** — ¿Responde a la intención del usuario? ¿Supera al competidor?
5. **Off-Page** — ¿Cuántos y qué calidad de backlinks tiene el sitio?
6. **Local** — ¿Está optimizado el GBP? ¿NAP consistente?
7. **Competencia** — ¿Qué hace el top 3? ¿Cuáles son sus keywords ganadoras?
8. **Plan de acción** — Lista priorizada de mejoras con impacto estimado.
